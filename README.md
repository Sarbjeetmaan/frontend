# MERN E-Commerce Website

A full-stack e-commerce web application built using the MERN stack. The application allows users to browse products, add items to the cart, place orders, make secure online payments, and track their orders. It also includes an admin panel for managing products and orders.

## Live Demo

[https://your-vercel-link.vercel.app](https://project-3v49.vercel.app/)

## Features

### User

- User Registration & Login
- JWT Authentication
- Product Search
- Shopping Cart
- Online Payment Integration
- Order Tracking
- Order History
- Product Reviews & Ratings
- Responsive Design
- Secure Online Payment (Cashfree)

### Admin

- Admin Dashboard
- Add, Update and Delete Products
- Manage Orders
- Upload Product Images using Cloudinary

## Tech Stack

**Frontend**
- React.js
- React Router
- Axios
- CSS 

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt
- Cloudinary
